## Оглавление
* [Начало](#начало)

## Начало
1. установить [Node.js](https://nodejs.org/en/);
2. ``npm i``;
3. готово.


## Сборка проекта
* `npm run dev`     — запустить среду для разработки;
* `npm run build`   — запустить среду для продакшена;
* `npm run preview` — посмотреть результат.
